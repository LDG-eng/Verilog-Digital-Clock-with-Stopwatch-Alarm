# Assets Overview

This directory stores all visual materials related to the project.

## 📂 Structure

- **images/**  
  Block diagrams, RTL hierarchy diagrams, and synthesis schematics.

- **waveforms/**  
  Simulation waveform captures used for functional verification.

## 📌 Notes

All images are generated from ModelSim simulation results or Vivado synthesis outputs.
These materials are referenced throughout the documentation (`docs/`) and README files.
